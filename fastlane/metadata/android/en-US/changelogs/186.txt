Android: Tella FOSS 2.11.0 - Released on February 3, 2025
Replaced Google Maps with OpenStreetMaps
Added Android CameraX support
Added support for privacy-preserving analytics. All analytics are disabled by default and opt-in.
PDFs can now be opened and viewed directly inside Tella
Edit media: users can crop and rotate images within Tella's encrypted container
Feedback: users can now share feedback, bug reports, or improvement suggestions with the Tella team directly from inside Tella
Added new features in the Tella Web connection, including the ability to download and view resources uploaded by admins
Implemented background encryption so users can continue collecting data, taking photos, videos, and audio, and importing files while the encryption process takes place.
Added new accessibility settings
Fixes to improve performance and security
